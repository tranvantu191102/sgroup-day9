Bài 1. Arrow function và anonymous function

Arrow function được ra mắt ở ES6 giúp viết function được ngắn gọn hơn.
Ví dụ: const myFunction = (a,b) => a*b;
Anonymous function là một function không có tên, thường được dùng trong các function khác, hay một hàm được return về thì cũng được gọi là anonymous function
Ví dụ: const myFunction = () => {return function(a,b) => a*b}
